# Go study

Project made to study golang

## Getting Started

Clone the project.

To clone the project:
```
git clone https://github.com/weslopes/go_study.git
```

Enter the project directory:

```
cd go_study
```

### TESTS

To run tests just run command:
```
go test
```

To see the details use:
```
go test -v
```
